# Análise exploratória (EDA) de Dados com Pandas Profiling

Neste projeto, é usada a ferramenta Pandas Profiling (atualmente
chamada ydata-profiling) para gerar automaticamente relatórios de análise exploratória
de dados (EDA).

### Características

- Linguagem: Python
- Biblioteca: Pandas Profiling, Streamlit, Matplotlib, Seaborn, Jinja2 
- Tipo de Análise: Análise Exploratória de Dados (EDA)
- Formatos de dataset: CSV
- Saída: Relatório interativo de EDA em página Web

### Manutenção

A página está sendo mantida através do Streamlit Community Cloud. 

Acesso: https://eda-analysis-project.streamlit.app/.
